# Abdulla Hashir Ali - Senior PHP/Laravel Developer Portfolio

A modern, responsive portfolio website showcasing PHP/Laravel development skills, ERP systems, and full-stack web applications.

## Getting Started

1. Install Dependencies

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Technologies Used

- Next.js 15
- TypeScript
- Tailwind CSS
- React
- Node.js

## Features

- Responsive design
- Dark/Light theme support
- Modern UI components
- SEO optimized
- Fast performance
